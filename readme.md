## Description

TorchCS Loader is a DLL/SO file that is loaded alongside Microsoft Detour.

## Required Tools

- **CMake 3.20+** (recommended)
- **A C++20 compiler:**
  - **Windows:** MSVC (Visual Studio 2022) or MinGW
  - **Linux:** GCC 11+ or Clang 14+
